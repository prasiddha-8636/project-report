\chapter{EXPECTED OUTCOMES}

The project is expected to result in a system that can take a user image and output a predicted geographic location and camera heading. The output will include a visual of the detected horizon line compared against the synthetic mountain shape from the database to show how the match was made. By providing a ranked list of the top candidate matches rather than a single point, the system can still be considered successful if the true location appears within the top-tier results, even when a perfect coordinate match is not achieved due to terrain similarities or image noise.
